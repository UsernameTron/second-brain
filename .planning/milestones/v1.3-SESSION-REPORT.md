## Key Learnings (from RETROSPECTIVE.md)

- **Cross-AI audit feeds concrete backlog** — no guessing about what to fix
- **In-phase CI ratchet** — moving the bar in the PR that earned the coverage prevents regression in that PR itself
- **Hybrid mock reduction** — new integration suite alongside existing unit tests preserves fast feedback and gains wiring confidence
- **Read coverage report before writing tests** — data beats scope-doc for prioritizing files
- **Run `CI=true` locally** — CI skips UAT and other guarded suites, so local runs without that flag are optimistic
