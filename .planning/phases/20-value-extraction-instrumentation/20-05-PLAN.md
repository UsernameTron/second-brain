Rules:

<!-- Definition of "delta" -->

The summary describes "yesterday's growth" relative to the day before. So:
- `+N proposals` = priorRow.proposals (proposals are a count of activity yesterday, not a balance ‚Äî emit verbatim with + sign)
- `+M promotions` = priorRow.promotions (same ‚Äî activity count)
- `+X.X KB memory` = (priorRow.memory_kb - dayBeforePrior.memory_kb), to 1 decimal, signed
  - If no day-before-prior row exists, fall back to priorRow.memory_kb formatted with `+`
- `+K entries` = (priorRow.total_entries - dayBeforePrior.total_entries), signed
  - If no day-before-prior row exists, fall back to priorRow.total_entries with `+`
- `R recalls` = priorRow.recall_count (count, no sign)

    ```javascript
    /**
     * Format the verbatim 5-delta "Yesterday: ..." summary line for the top of /today briefings.
     * Pure function ‚Äî no I/O. Returns empty string on day-1 or missing priorRow.
     *
     * @param {object|null} priorRow - The row for yesterday, shape { proposals, promotions, total_entries, memory_kb, recall_count }
     * @param {object|null} [dayBeforePrior] - The row for the day before yesterday, same shape; used for memory_kb and total_entries deltas
     * @returns {string} Verbatim summary line, or '' when priorRow is null/undefined
     */
    function buildYesterdaySummaryLine(priorRow, dayBeforePrior) {
      if (!priorRow) return '';

      const sign = (n) => (n >= 0 ? `+${n}` : `${n}`);
      const signKb = (n) => {
        const rounded = Math.round(n * 10) / 10;
        return rounded >= 0 ? `+${rounded.toFixed(1)}` : `${rounded.toFixed(1)}`;
      };

      const proposals = sign(priorRow.proposals || 0);
      const promotions = sign(priorRow.promotions || 0);

      const memoryDelta = dayBeforePrior
        ? (priorRow.memory_kb || 0) - (dayBeforePrior.memory_kb || 0)
        : (priorRow.memory_kb || 0);
      const memoryStr = signKb(memoryDelta);

      const entriesDelta = dayBeforePrior
        ? (priorRow.total_entries || 0) - (dayBeforePrior.total_entries || 0)
        : (priorRow.total_entries || 0);
      const entriesStr = sign(entriesDelta);

      const recalls = priorRow.recall_count || 0;

      return `Yesterday: ${proposals} proposals, ${promotions} promotions, ${memoryStr} KB memory, ${entriesStr} entries, ${recalls} recalls`;
    }

    module.exports = { ...existingExports, buildYesterdaySummaryLine };
    ```
