    Create hooks/post-merge-doc-sync.js as a CJS module:
    - Export `extractDocStats(text)` — uses regex to extract testCount, coverageStatements, coverageBranches from markdown text. Patterns:
      - Test count: `/(\d[\d,]*)\s+total(?:\s+across|\b)/i` — parse result with parseInt after removing commas
      - Statement coverage: `/Statements?\s+([\d.]+)%/i` — parseFloat
      - Branch coverage: `/Branch(?:es)?\s+([\d.]+)%/i` — parseFloat
    - Export `compareStats(docStats, liveStats, warnThreshold)` — compares each stat pair; testCount uses exact match (threshold 0); coverage stats use warnThreshold. Returns array of violation strings, each formatted as: `"${label}: doc states ${docVal}, actual is ${liveVal}, drift=${drift.toFixed(2)}"`.
    - Export `getLiveStats(projectRoot)` — runs `execFileSync('npx', ['jest', '--coverage', '--json', '--outputFile=' + tmpOut, '--silent', '--forceExit'], { cwd: projectRoot, timeout: 60000, stdio: ['ignore', 'ignore', 'ignore'] })`. Reads tmpOut for numTotalTests. Reads coverage/coverage-summary.json for statement/branch pct. Returns `{ testCount, coverageStatements, coverageBranches }` or null on failure. Always cleans up temp file.
    - Export `main()` — reads CLAUDE.md and README.md from project root (path.join(__dirname, '..')), loads config/docsync.json for threshold, calls getLiveStats, calls extractDocStats on each doc, calls compareStats for each doc. Prints warnings to stderr if violations found. Prints "[post-merge] Documentation stats are current." if clean. Wraps EVERYTHING in try/catch and calls process.exit(0) — NEVER exits non-zero.

    ```bash
    #!/usr/bin/env bash
    # Post-merge hook: documentation drift warning (non-blocking).
    # CRITICAL: This hook must NEVER exit non-zero. A non-zero exit
    #           can abort or confuse git merge operations.
    SELF="$(readlink -f "$0" 2>/dev/null \
          || python3 -c "import os; print(os.path.realpath('$0'))")"
    HOOK_DIR="$(dirname "$SELF")"

    echo "[post-merge] Checking documentation drift..."
    node "$HOOK_DIR/post-merge-doc-sync.js" || {
      echo "[post-merge] Warning check failed (non-blocking)" >&2
    }
    exit 0
    ```
